# ModernTech HR Management System

## Project Description

ModernTech HR Management System is a Vue 3 and Bootstrap proof-of-concept application designed to help HR staff manage employee information, payroll, attendance tracking and leave requests from a centralised system.

## Features

- Employee Management
- Employee Search
- Add Employee Functionality
- Delete Employee Functionality
- Payroll Calculations
- Digital Payslip Generation
- Attendance Tracking
- Leave Request Approval and Denial
- Responsive Bootstrap Interface

## Technologies Used

- Vue 3
- JavaScript
- Bootstrap 5
- Vite

## Installation

npm install

npm run dev
